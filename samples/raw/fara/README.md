# FARA Foreign Agent Registration

- **Source:** US Dept. of Justice, National Security Division
- **Full download:** `https://efile.fara.gov/bulk/zip/FARA_All_Registrants.csv.zip`
- **Catalog entry:** [CATALOG.md — FARA](../../../CATALOG.md#fara-foreign-agent-registration)
- **License:** US government public domain

## Files

### registrants-sample.csv

CSV with header row. Columns:

| Column              | Description                                  |
|---------------------|----------------------------------------------|
| Registration Number | Unique FARA registration number              |
| Registration Date   | Date registered (MM/DD/YYYY)                 |
| Termination Date    | Date terminated (empty if active)            |
| Name                | Registrant name (person or firm)             |
| Business Name       | Business/firm name                           |
| Address 1           | Street address line 1                        |
| Address 2           | Street address line 2                        |
| City                | City                                         |
| State               | State (2-letter code)                        |
| Zip                 | ZIP code                                     |

## Entity types

Persons and organizations registered as agents of foreign principals in
the United States.
